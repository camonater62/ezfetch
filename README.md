EZFetch is an simple infogathering script for screenshots on your UNIX system!

## DOWNLOAD
git clone -b compact https://github.com/camonater62/ezfetch

## USAGE w/o INSTALL
cd ezfetch
chmod +x ezfetch
./ezfetch

## INSTALLATION
cd ezfetch
chmod +x ezfetch
sudo cp ezfetch /usr/bin

## USAGE
ezfetch

## DEPENDENCIES
uname
lsb_release
xprop
free
lspci
